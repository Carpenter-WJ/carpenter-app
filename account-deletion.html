‹ 현장일지로 돌아가기

계정 삭제 안내

현장일지(운영: 직툴마켓 · 대표 송원준)

현장일지는 앱 안에서 직접 계정과 모든 데이터를 삭제할 수 있는 기능을 제공합니다. 별도 웹사이트 방문이나 이메일·전화 문의 없이 앱 내에서 바로 처리됩니다.

1. 삭제 절차

  1. 현장일지 앱 실행 후 로그인
  2. 하단 탭에서 설정 이동
  3. 맨 아래 로그아웃 버튼 밑에 있는 "계정 삭제" 링크 선택
  4. 안내 문구를 확인하고 2단계 확인(팝업 2회)을 거쳐 삭제 진행
  5. 확인 즉시 계정과 데이터가 영구적으로 삭제되며, 자동으로 로그아웃됩니다

2. 삭제되는 데이터 및 보관 기간

삭제 요청 시 아래 데이터가 지체 없이(즉시) 영구 삭제되며, 별도로 보관되는 기간은 없습니다.

구분삭제 대상
계정 정보이름, 이메일, 프로필 사진, 로그인 계정 정보 전체
작업 기록현장명·작업일자·일당·작업 사진·현장 메모 전체
정산 기록입금 내역, 정산 상태, 계좌번호
팀 정보본인이 팀장인 경우 팀 자체가 해체되며(팀원 기록은 각자 개인 기록으로 자동 이관), 팀원인 경우 해당 팀에서만 제거됩니다
구매 내역프리미엄 결제 확인 기록(단, 전자상거래법에 따라 결제 사실 자체는 5년간 별도 보관될 수 있습니다)
※ 결제(전자상거래법), 세무 등 관계 법령에 따라 보존 의무가 있는 최소한의 거래 기록을 제외한 모든 개인정보와 서비스 이용 기록은 삭제 요청과 동시에 즉시 파기됩니다.

3. 계정을 삭제하지 않고 일부 데이터만 삭제하고 싶다면

계정은 유지하면서 특정 작업 기록, 정산 내역, 작업 사진만 개별적으로 삭제하는 것도 가능합니다. 각 항목의 삭제 버튼(🗑)을 눌러 개별 삭제할 수 있습니다.
